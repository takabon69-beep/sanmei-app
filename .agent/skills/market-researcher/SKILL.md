# market-researcher

市場環境や競合状況を構造化されたフレームワーク（3C, SWOT, PEST）を用いて分析します。

## Instructions

1.  **分析手法の選択**: 調査目的に応じて以下のフレームワークから適切なものを選択または組み合わせて使用します。
    - **3C分析**: Customer（市場・顧客）, Competitor（競合）, Company（自社）の観点で分析。
    - **SWOT分析**: Strength（強み）, Weakness（弱み）, Opportunity（機会）, Threat（脅威）の観点で分析。
    - **PEST分析**: Politics（政治）, Economy（経済）, Society（社会）, Technology（技術）の観点で分析。

2.  **情報収集**: ブラウザツールを使用して、最新の市場データ、ニュース、競合他社の情報を収集します。

3.  **分析の実施**:
    - 収集した情報を各フレームワークの項目に分類します。
    - 事実（Fact）に基づいた客観的な分析を心がけます。
    - 各項目間の関連性や、示唆（Insight）を抽出します。

4.  **アウトプット**: 構造化された形式で分析結果を提示します。

## Output Example

```markdown
# [案件名] 市場調査レポート

## 1. PEST分析（マクロ環境）
- **Politics**: [規制緩和、法改正など]
- **Economy**: [市場成長率、為替など]
- **Society**: [ライフスタイルの変化、人口動態など]
- **Technology**: [新技術の台頭、DXなど]

## 2. 3C分析（ミクロ環境）
- **Customer**: [ターゲット層、ニーズの変化]
- **Competitor**: [主要競合のシェア、戦略]
- **Company**: [自社のリソース、独自の提供価値]

## 3. SWOT分析（内的・外的要因）
- **Strength**: [自社が勝っている点]
- **Weakness**: [課題、不足している点]
- **Opportunity**: [追い風となる外部環境]
- **Threat**: [逆風となる外部環境]

## 4. 総合的な示唆
- [分析から導き出される今後のアクション案]
```
