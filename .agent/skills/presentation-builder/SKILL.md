# presentation-builder

ビジネスや公的な場で「伝わる」プレゼンの構成案を作成します。

## Instructions

1.  **目的の確認**: ターゲット、ゴール、持ち時間を確認。
2.  **構成の選択**:
    - **PREP法**: 結論→理由→例→結論（説得・提案など）
    - **SDS法**: 要約→詳細→要約（報告・説明など）
    - **三段構成**: 序論→本論→結論
3.  **スライド案作成**: 各スライドに「タイトル」「キーメッセージ（要点1行）」「内容（箇条書き）」「ビジュアル案」を記述。
4.  **推敲**: 1スライド1メッセージ、結論ファースト、平易な言葉を徹底。

## Output Example

```markdown
# [タイトル]
## 1. 結論
- **キーメッセージ**: [一番伝えたいこと]
- **内容**: 
    - [詳細1]
    - [詳細2]
- **ビジュアル**: [グラフ/写真の指示]
```
