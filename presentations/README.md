# プレゼンテーション作成ガイド

このフォルダは、Antigravity（AIエージェント）と共にスライド資料を作成するための作業スペースです。

## ワークフロー

### 1. スライド構成案の作成
AIに作りたいテーマを伝えてください。スキル「`presentation-builder`」を使用して、以下の要素を含むMarkdown形式の構成案を作成します：
- 各スライドのタイトル
- キーメッセージ（一言で伝わる要点）
- 詳細内容（箇条書き）
- ビジュアル案（図解や写真の指示）

### 2. パワーポイント（.pptx）ファイルの生成
構成案が確定したら、AIに「パワーポイントファイルを生成して」と指示してください。Pythonの `python-pptx` ライブラリを使用して、実際に開けるスライドファイルをこのフォルダ内に出力します。

### 3. 素材の準備
スライドに挿入する画像やイラストが必要な場合は、AIに「スライド1用の画像を生成して」と指示してください。`generate_image` を使用して最適な画像を生成します。

## フォルダ構成のヒント
複数のプロジェクトを同時進行させる場合は、以下のようにプロジェクトごとのフォルダを作成することをお勧めします。
- `presentations/project-alpha/`
- `presentations/project-beta/`

## ブランド・デザイン規定

今後作成するすべてのプレゼンテーションは、以下のブランディング・ルールを標準として適用します。

### 1. ロゴの配置
- **配置箇所**: 各スライドの **左上**
- **素材**: `presentations/brand/logo.png` を参照します。

### 2. コピーライト（フッター）
- **表記**: `(C) 2026 株式会社村上経営研究所`
- **配置箇所**: ページ下部の **中央**。

### 3. ページ番号
- **配置箇所**: スライドの **右下**
- **表示形式**: 現在のページ番号を表示。

### 4. 保存先とファイル名
- **保存フォルダ**: `presentations/outputs/`
- **命名規則**: `YYYYMMDD_タイトル.pptx` （例：20260406_新入社員研修.pptx）
